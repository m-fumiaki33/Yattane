# Yatta - 赤ちゃんの成長記録アプリ

「Yatta（やった！）」は、子どもの成長の瞬間（マイルストーン）を簡単・シンプルに記録できるiOSアプリケーションです。
初めての「寝返り」や「たっち」、「パパ・ママ」と言えた日など、大切な思い出を写真と共に残しましょう。

## 主な機能

*   **成長の記録**: タイトル、日付、写真、メモを入力して、子どもの成長を記録できます。
*   **カレンダー表示**: カレンダー形式で記録を振り返ることができます。
*   **複数の子どもに対応**: 兄弟・姉妹ごとのプロフィールを作成し、切り替えて管理できます。
*   **年齢・月齢の自動表示**: 誕生日から計算した、その記録時点での月齢を表示します。
*   **データの管理**: 記録や子どものプロフィールの編集・削除も簡単に行えます。

## 技術スタック

*   **言語**: Swift
*   **UIフレームワーク**: SwiftUI
*   **データベース**: Core Data
*   **ターゲット**: iOS 17.0+

## 開発環境

*   Xcode 15+
*   Swift 5.9+

## 使い方

1.  アプリを起動し、子どものプロフィール（名前、誕生日、性別）を登録します。
2.  「記録する」ボタンから、新しいマイルストーンを追加します。
3.  左上のメニューから、表示する子どもを切り替えたり、プロフィールの編集・削除が行えます。
4.  下のタブで「記録（リスト）」と「カレンダー」を切り替えられます。
