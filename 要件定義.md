# Yattane (やったね！) 要件定義書

## 1. アプリ概要
「Yattane（やったね！）」は、子どもの成長の瞬間（マイルストーン）を簡単・シンプルに記録できるiOSアプリケーションです。
初めての「寝返り」や「たっち」、「パパ・ママ」と言えた日など、大切な思い出を写真と共に残すことを目的としています。

## 2. ターゲットユーザー
- 子育て中のパパ・ママ
- 子供の成長を手軽に記録・振り返りたい方
- 複数の子どもの記録を1つのアプリで管理したい方

## 3. システム要件（技術スタック・開発環境）
- **対象OS**: iOS 17.0 以上
- **開発言語**: Swift 5.9 以上
- **UIフレームワーク**: SwiftUI
- **データベース・同期**: Core Data + CloudKit (iCloudを利用した複数端末間のデータ同期およびバックアップ)
- **統合開発環境 (IDE)**: Xcode 15 以上

## 4. 機能要件

### 4.1. ユーザー（子ども）プロフィール管理機能
- **プロフィール作成**: 名前、誕生日、性別の登録。（プロフィール用の顔写真アイコン設定は不要）
- **プロフィール編集・削除**: 登録済みのプロフィール情報の変更、および削除。
- **プロフィール切り替え**: ホーム画面左上のメニューから、表示する子どもの記録を切り替え可能。

### 4.2. 成長記録（マイルストーン）管理機能
- **記録の追加**: 以下の情報を入力して記録を作成する。
  - タイトル（例：「初めての寝返り！」）
  - 日付
  - 写真（**複数枚**の添付が可能。横スクロール等で閲覧。動画記録は非対応）
  - メモ
- **月齢自動計算**: プロフィールに登録された誕生日と記録日の情報から、記録時点での「年齢・月齢（例：1歳2ヶ月）」を自動計算して表示。
- **記録の編集・削除**: 作成した成長記録の修正・削除。
- **シェア機能**: セキュリティ・プライバシー上の観点から、SNSシェア等の機能は不要（持たせない）。

### 4.3. 表示機能（タブ切り替え）
画面下部のタブバーで以下の2つのビューを切り替える。
- **リスト表示タブ**: 記録を時系列（降順）で一覧表示。
- **カレンダー表示タブ**: カレンダー形式で記録を表示。
  - 同じ日に複数の記録がある場合、カレンダーの日付枠に複数の点（ドット）を表示する。
  - 日付をタップすると、その日の記録がモーダル（またはリスト）で立ち上がる。

### 4.4. 通知（リマインド）機能
- 「もうすぐハーフバースデーですね！」「1歳のお誕生日おめでとうございます！」等の記念日を知らせる通知（ローカル通知またはプッシュ通知）。

## 5. 画面構成（スクリーン一覧・案）

1. **初回起動/初期設定画面**
   - 初回起動時の子どもプロフィール登録画面。
2. **ホーム（リスト）画面**
   - 選択中の子どもの記録をリスト表示。
   - 画面左上にプロフィール切り替え・設定メニューへのアクセスボタン。
   - 画面内に「記録する（＋）」ボタン。
3. **カレンダー画面**
   - カレンダー上の日付に記録の有無を点（ドット）で表示。
   - 日付をタップでその日の記録詳細をモーダル表示。
4. **記録追加/編集画面**
   - 写真選択（複数枚）、タイトル、日付、メモの入力フォーム。
5. **プロフィール一覧・設定画面（左上メニュー）**
   - 登録済みの子ども一覧と切り替え。
   - 子どもの追加、編集、削除。

## 6. 非機能要件
- **シンプルで直感的なUI**: 忙しい育児中でも素早く記録できるよう、入力ステップを最小限にする。
- **オフライン動作及同期**: 基本的な操作はエッジ（オフライン）で可能としつつ、バックグラウンドでCloudKitを通じてシームレスに同期されること。
- **プライバシー**: 外部へのシェア機能を持たせず、安全なデータ保管を担保する。
